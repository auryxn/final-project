# Network Labs Portfolio 🌐
Professional Cisco Packet Tracer laboratory works.

## 📁 Labs Overview
1. **Lab 2: Introduction to Packet Tracer** - Basic topology, IP addressing, and connectivity.
2. **Lab 4: Static Routing** - Configuring static routes in a triangle topology (3 routers).
3. **Lab 5: VLAN & Inter-VLAN Routing** - VLAN configuration and Router-on-a-Stick.
4. **Lab 6: DHCP, DNS & HTTP Servers** - Automating IP assignment and hosting web services.
5. **Lab 8: NAT (Network Address Translation)** - Static NAT, PAT, and Port Security.
6. **Lab 9: IPsec VPN** - Secure site-to-site communication using IPsec tunnels.

## 🛠️ Tools
- Cisco Packet Tracer 8.x
- CLI Configuration (IOS)
