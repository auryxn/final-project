# Lab 6: DHCP, DNS & HTTP Servers

## 🎯 Objective
Configure network services to automate IP assignment and host web resources.

## 🛠️ Services Setup
1. **DHCP:** Pool configuration, DNS server assignment, Default Gateway.
2. **DNS:** Map domain names (e.g., `sir.com`) to IP addresses.
3. **HTTP:** Host and modify web pages.

## ✅ Workflow
- Client PC requests IP via DHCP.
- Client browser accesses HTTP server using DNS name.
